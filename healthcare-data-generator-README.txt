healthcare-data-generator-README

1. Generates realistic healthcare data with:
   - Multiple departments and resource types
   - Time-based patterns (day/night variations)
   - Seasonal patterns
   - Department-specific patterns
   - Realistic usage distributions

2. Includes a test function that demonstrates:
   - Data generation
   - Pattern analysis
   - Resource optimization
   - Scheduling functionality
   - Report generation

The generated data includes patterns like:
- Higher emergency department activity at night
- Increased surgery department usage during weekdays
- Seasonal variations in utilization
- Realistic resource availability windows
