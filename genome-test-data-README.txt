genome-test-data-README

1. Test Data Generation Functions:
   - `generate_random_dna()`: Creates random DNA sequences with controllable GC content
   - `generate_sequence_with_motifs()`: Inserts specific motifs into sequences
   - `create_test_fasta()`: Generates a FASTA file with various test sequences

2. Testing Function:
   - `test_genome_analyzer()`: Comprehensive test of the GenomeAnalyzer package
   - Creates test data and runs analysis
   - Prints results for verification

The test data includes:
- Sequences with high GC content
- Sequences with inserted motifs
- Random sequences with normal GC content
