genome-analyzer-viz-README

1. Statistical Validation:
   - Mean, standard deviation, median calculations
   - Skewness and kurtosis analysis
   - Shapiro-Wilk test for normality
   - Statistical summary reports

2. Visualizations:
   - Feature distribution plots with histograms and KDE
   - Windowed analysis plots showing GC content and complexity variation
   - Publication-quality figures using seaborn styling

3. Enhanced Analysis Pipeline:
   - Automated report generation
   - Organized output directory structure
   - Comprehensive statistical summaries

The code creates:
- Distribution plots for each feature
- Time-series style plots for windowed analysis
- Detailed statistical reports
- All outputs are saved in an organized directory structure
